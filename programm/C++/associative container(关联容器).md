## 类型
有序:
map                                                关联数组：关键字-值 对
set                                                   只保存关键字
multimap                                        关键字可重复出现的map
multiset                                           关键字可重复出现的set

无序:
unordered_map                             用哈希函数组织的map
unordered_set                                用哈希函数组织的set
unordered_multimap                     关键字可重复出现的unordered_map
unordered_multiset                       关键字可重复出现的unordered_set
